# About Page — Image Reference

All images live in `public/images/about/`. New candidate images sourced from
`splint_business_research/260604_product_photos/` were resized to 1920px longest
side at 82% JPEG quality before copying (BuddyRing originals were 3000–4000px, 2.4–2.9MB).

Existing images are all 1920×1280px, 200–600KB. Vercel's image optimizer serves
appropriately sized versions in production based on the `sizes` prop on each `<Image>`.
